# word-search-generator

A web app that generates word search puzzles, built using React and NextJS.

## Setup

Before you begin, you'll need to install the dependencies:

```bash
npm install
```

## Running the App

To start the NextJS server, run:

```bash
npm run dev
```

Then point your browser to [http://localhost:3000](http://localhost:3000) to see the app.
